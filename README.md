# aws-ec2-nginx-static-website
Hands-on cloud project demonstrating static website hosting on AWS EC2 using Nginx and Linux.

Static Website Deployment on AWS EC2 using Nginx

 Project Overview
This project demonstrates how to deploy a static website on an AWS EC2 (Linux) instance using Nginx as the web server.
The main objective was to gain hands-on experience with cloud infrastructure, Linux server management, and real-world website hosting.

 Tech Stack
AWS EC2 (Linux Instance)
Nginx Web Server
HTML & CSS
Linux Terminal
Vim Editor
Public IP-based Hosting

 Deployment Steps
Launched an AWS EC2 Linux instance
Connected to the server using SSH
Installed and configured Nginx
Uploaded static website files (index.html)

Set proper file permissions
Started and enabled Nginx service
Accessed the live website using EC2 Public IP

Live Deployment
The website was successfully hosted and accessed via the EC2 public IP address.
